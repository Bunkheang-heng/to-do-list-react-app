# Event Planning Portal

This app is intended to help you organize your tasks so you can be more organized in your daily life. Whether it would be for work or for day to day activities. 

The event planning app requires an account to use. This is done to ensure that the web app can be used online with its own data for different users. Once an account is made, several new functions are available to the user.

## Home

The Home page is a summary of tasks. It gives the user a quick access to any overdue tasks as well as automatically sorting tasks by priority.

## Tasks

The Tasks page is where task creation happens. This page allows the user to create new tasks. The create tasks button allows the user to create a new tasks with a short description and a priority.

In addition to allowing the user to check off tasks. The Tasks page also filters out Completed tasks from Active tasks so the user can delete Completed tasks if they no longer wish to keep it there for archival purposes.

## Profile

The Profile page lists the user's name, email, id, and join date. This page has a shortcut to the settings as well as a log out button.

## Settings

The Settings page allows the user to change their passwords, the website theme (for those who prefer dark mode) and allows the user to delete their account.

Account deletion requires a password in order to prevent users from accidentally deleting their accounts.
